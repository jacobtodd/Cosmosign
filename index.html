✓ Verify a Signature

Cryptographically verify a message was signed by a specific Cosmos address. All fields are required.

ℹ How it works
Signing — Keplr signs your message using ADR-036, the Cosmos off-chain signing standard. Your message is wrapped in a canonical amino document and signed with your wallet's secp256k1 private key.

Verification — The secp256k1 signature is verified entirely in your browser using the elliptic library. The signer's address is cryptographically derived from the public key using SHA-256 → RIPEMD-160 → bech32, proving the signature genuinely came from that address.
